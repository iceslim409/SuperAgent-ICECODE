🚀 What ICECODE Can Do

        

          

            
🤖 Multi-Agent Swarm

            
Pipeline & parallel orchestration. Multiple AI agents collaborate on a single task. Supervisor pattern with sub-agent delegation.

          

          

            
📚 Local RAG Knowledge Base

            
Semantic search over your documents. Index .txt, .md, .py, .pdf, .json. 100% local — sentence-transformers + FAISS, no cloud needed.

          

          

            
🔌 34 Platform Gateways

            
Slack, Discord, Telegram, WhatsApp, Email, GitHub, Jira, Notion, Twitter, LinkedIn, and 24 more. Multi-channel messaging in one place.

          

          

            
⚡ WebSocket Real-Time Chat

            
Bidirectional streaming. Cancel generation mid-stream. SSE fallback. Works with all 14 LLM providers simultaneously.

          

          

            
🎯 Goals & Ralph Loop

            
Long-horizon goal pursuit. Agent autonomously breaks goals into tasks, executes, self-evaluates, and improves over time.

          

          

            
🧠 Self-Learning Skills (RL)

            
Reinforcement learning training environment. Agent learns from feedback, saves skills, improves without retraining the model.

          

          

            
🖥️ Desktop App (Electron)

            
Native desktop app for Linux, Windows, macOS. Wraps the React UI. Auto-connects to local Python backend on port 13210.

          

          

            
💻 TypeScript CLI (TUI)

            
Terminal user interface powered by Ink. Auto-detects Python backend. 11 TypeScript packages including React UI, SDK, tools, MCP, gateway.

            FastAPI

            SolidJS + React
